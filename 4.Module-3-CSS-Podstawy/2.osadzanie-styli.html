Osadzanie styli w HTML'u

Pierwszy sposób

Style można dodawać w pliku .html za pomocą znacznika 

 <style> </style>
zamieszczonegow znaczniku 
<head> </head>
dokumentu html - nie jest to jednak metoda zalecana w obecnych czasach z powodu niepotrzebnego obciążania dokumentu html

Drugi sposób

Style można implementować/osadzać inlineow'o w danym tagu, co czyni je najmocniejszymi w hierarhi implementacjia, przkład: < p style="color: orangered;" >Nasz tekst < /p> Co da nam następujący efekt: 

Nasz tekst
co jednak również nie jest zalecane przy tworzeniu pliku HTML, a można to zastąpić za pomocą atrybutu !important w kodzie CSS przy wartości atrybutu, powodując go byciem najważniejszym ze wszystkich styli i nienadpisywalnym, do jednak również stosuje się bardzo rzadko ze względu na ograniczenia które to on powoduje przy updatowaniu/rozwijaniu stron HTML.

Trzeci sposób

Przyjętym za najlepszy sposób jest stylizowanie elementów html w osobnym pliku, a następnie dołączenia do znacznika head linku do tego pliku, np: 

< link rel="stylesheet" href="./styles/2.osadzanie-styli.css" />
            gdzie:
                rel - jest to rodzaj powiązanie/relationship z naszym dokumentem html, a  stylesheet jest to
            href - odpowiada za ścieżkę do tego pliku - tutaj pokazana jest tylko przykładowa ścieżka.
